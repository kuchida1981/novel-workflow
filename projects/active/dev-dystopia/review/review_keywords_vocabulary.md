# キーワード語彙レビュー — 現代語・技術語の使用について

> **対象:** `05_draft/` 全章  
> **日付:** 2026-04-24  
> **観点:** ファンタジー世界観と現代技術語彙の不釣り合い

---

## 発端となった観察

`05_draft/` 全体を通読した結果、以下のキーワードが**世界観との不釣り合い**を感じさせる一方で、**ニュアンスとしては正しい**という二重の印象を持つことが確認された。

```
演算 / 資本 / 市場 / 消費する / 回路 / エネルギー / 論理 / 構築 /
エンジニア / ノイズ / リフレッシュ / 点灯 / グラフ / リアルタイム /
データ / コピー / 統計 / 平均的な最適化 / システム / コスト / 最適化
```

---

## 判断の枠組み

この語彙群を一律に「除去すべきミス」と見なすのは誤り。  
**「良い緊張」と「余計な緊張」** に分類して評価する。

---

## 良い緊張として機能するもの（意図的な仕掛け）

これらは不釣り合いに見えて、**物語の構造・テーマ上必要な効果**を担っている。

| キーワード | 機能・理由 |
|---|---|
| **回路** | ドレスの縫い目＝回路という比喩は作品の核。変えると世界観ごと崩れる |
| **演算** | リネットが「感じる人間」から「計算する存在」へと変容していく描写として機能 |
| **データ / コピー** | 男爵がリネットを人間ではなくデータとして扱う残酷さを、語彙レベルで可視化する |
| **統計 / 平均的な最適化** | scene 2-3の核心台詞。機械との差異を言語化するためにあえてこの語が必要 |
| **資本 / 市場** | 漢字表記のため浮かない。経済的構造の露出として自然に機能 |
| **ノイズ** | エルザの存在＝ノイズという倒置。リネットの人間性の喪失を暗示 |
| **システム** | 「巨大なシステムを銀の針で突き崩す」——抽象的な支配構造の表現として有効 |
| **コスト** | 漢字ではないが日本語に十分定着しており、経済論理の浸透を示す語として機能 |

**根拠:** `core.md` に「大人になったとき、効率化と人員削減の嵐に直面した際にこの物語を思い出してほしい」とある。読者が無意識に現代と接続できる語彙は**意図的に残すべき**であり、一律排除はテーマを損なう。

---

## 余計な緊張として見直す余地があるもの

ニュアンスは正しいが、**ファンタジーの皮が剥がれすぎる**リスクがあるもの。

| キーワード | 問題点 | 代案候補 |
|---|---|---|
| **エンジニア** | 最も露骨。「職種名カタカナ」として際立つ。2026年代IT業界の響きが強い。scene 1-2 / scene 2-3 の2箇所に登場 | 職人・技師（ただし「エンジニアとしての誇り」が持つ現代的強度は下がる） |
| **リアルタイム + グラフ** | scene 1-3 L12「魔力消費グラフがリアルタイムで描かれている」——2語が並ぶとIT文脈が過剰になる | グラフ→軌跡 / リアルタイム→刻々と・その場で |
| **リフレッシュ** | scene 1-2 L51「リフレッシュされたドレスの回路」。コンピュータ的な用法であり文脈で浮く | 更新された・刷新された |

---

## 総合判断

### 「不釣り合い」はバグではなくフィーチャー

この語彙群は**作品の批評的な力の源泉**でもある。一律除去ではなく、「どこで意識させるか」を設計的に選ぶことが重要。

### 優先度の高い対処

1. **`エンジニア` の統一判断が最優先。**  
   他の語は漢字・技術一般語として背景に溶け込めるが、これだけが「特定職種カタカナ」として際立つ。  
   - 選択肢A: 全箇所を「職人」「技師」に戻し、現代感を抑える  
   - 選択肢B: 1箇所だけ意図的に残し、他は職人に統一する（意識させる箇所を絞る）

2. **`リアルタイム + グラフ` の並置を解消。**  
   単独なら許容範囲だが、scene 1-3の1文に2語同時は過剰。どちらか一方を代案に換える。

3. **`リフレッシュ` は単純に言い換え可。**  
   意味は変わらず、浮きだけ解消できる。
