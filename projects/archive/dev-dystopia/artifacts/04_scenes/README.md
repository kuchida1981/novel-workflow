# Step 4: シーン (Scenes)

> **依存:** `03_plot/plot.md` (approved)

## 構成

各章をサブディレクトリで管理します。

```
04_scenes/
  chapter-01/
    scene-01.md
    scene-02.md
  chapter-02/
    scene-01.md
  ...
```

## シーンファイルのテンプレート

```markdown
# シーン: [章]-[番号] [タイトル]

> status: draft | approved
> 依存: plot.md — [対応する章のサマリー]

## 目的
（このシーンで達成すること）

## 登場人物
- （名前）: （このシーンでの役割・状態）

## 場所・時間

## 要点・ビート
1.
2.
3.

## 感情曲線
（シーン開始時の主人公の感情 → 終了時の感情）

## 次のシーンへの接続
```

## 承認チェックリスト（章単位）

- [ ] 全シーンが Plot の章サマリーをカバーしている
- [ ] 各シーンに明確な目的がある
- [ ] シーン間の接続が自然
