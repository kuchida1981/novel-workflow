# 世界観設定：星屑の社交界

## 1. 魔法のドレスと「星の雫（ほしのしずく）」
この世界の美しさは、天から降る希少な魔法の燃料「星の雫」によって駆動される。

- **星の雫 (Star Drops):** 眩しく輝く銀色の液体。極めて高価であり、一滴（ドロップ）単位で富豪たちの間で取引される。
- **魔法のドレス:** この雫を燃料として、着用者の容姿、声、振る舞いまでも「完璧」に補正・発光させるドレス。
- **維持コスト（燃費）:** ドレスを美しく光らせ続けるには、常に雫を注ぎ込み続けなければならない。例えば、一流のドレスなら「一晩の舞踏会で100滴」といった具合に。雫が尽きれば、ドレスはただの汚れたボロ布に戻り、魔法も解ける。

## 2. お針子（ニードルワーカー）の役割
ドレスの「構造」と「燃費」を管理する専門職。

- **役割:** 限られた「星の雫」をいかに効率よくドレスの各部位へ分配し、少ない雫で長く、美しく光らせるかを設計する。
- **「工夫」の価値:** 優れたお針子は、独自の縫製技術（魔法の回路設計）によって、通常の半分以下の雫で同等の輝きを実現できる。
- **現代の変化:** 資本力のある貴族は、最新の「自動織機（AI）」を導入し始めた。それは人間のお針子の「工夫」をデータとして吸い込み、誰でも・どこでも・自動で低燃費なドレスを作れるようにしていく。

## 3. 社会構造
- **上層貴族:** 莫大な「星の雫」を保有し、湯水のように雫を消費して常に最新のドレスを身にまとう。
- **没落貴族・中産階級:** 少ない「星の雫」をやりくりし、お針子の「工夫」に頼って社交界での地位を維持しようとする。
- **お針子ギルド:** 誇り高き職人集団だが、自動織機の普及により、その技術が「単なる効率化の部品」へと解体されつつある。

## 4. 経済の冷徹なルール
「星の雫」の価格は高騰し続けている。
貴族たちはドレスを光らせ続けるための費用を捻出するため、以下のどちらかを選択する。
1. **「更なる美しさ」の追求:** お針子の工夫で浮いた雫を使い、さらなる装飾（追加機能）を加えて、より高い地位（売上）を得る。
2. **「人件費の削減」:** お針子の工夫によって雫が節約できたら、そのお針子を解雇し、お針子の給金分でさらに「星の雫」を買い足す。
