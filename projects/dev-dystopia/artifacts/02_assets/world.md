# 世界観設定：星屑の社交界

## 1. 魔法のドレスと「星屑（スターダスト）」
この世界の美しさは、天から降る希少な資源「星屑」によって駆動される。

- **星屑 (Stardust):** 莫大なエネルギーを秘めた発光物質。極めて高価であり、富の象徴。
- **魔法のドレス:** 星屑を燃料として、着用者の容姿、声、振る舞いまでも「完璧」に補正・発光させるドレス。
- **維持コスト:** ドレスを美しく光らせ続けるには、常に星屑を供給し続けなければならない。供給が止まれば、ドレスはただの汚れたボロ布に戻り、着用者の魔法も解ける。

## 2. お針子（ニードルワーカー）の役割
ドレスの「構造」と「燃費」を管理する専門職。

- **役割:** 限られた星屑をいかに効率よくドレスの各部位へ分配し、長く、美しく光らせるかを設計する。
- **「工夫」の価値:** 優れたお針子は、独自の縫製技術（アルゴリズム的な最適化）によって、通常の半分以下の星屑で同等の輝きを実現できる。
- **現代の変化:** 資本力のある貴族は、最新の「自動織機（AI）」を導入し始めた。それは人間のお針子の「工夫」をデータとして吸い込み、誰でも・どこでも・自動で低燃費なドレスを作れるようにしていく。

## 3. 社会構造
- **上層貴族:** 莫大な星屑を保有し、常に最新のドレスを身にまとう。
- **没落貴族・中産階級:** 少ない星屑をやりくりし、お針子の「工夫」に頼って社交界での地位を維持しようとする。
- **お針子ギルド:** 誇り高き職人集団だが、自動織機の普及により、その技術が「単なる効率化の部品」へと解体されつつある。

## 4. 経済の冷徹なルール
星屑の価格は高騰し続けている。
貴族たちはドレスを光らせ続けるコストを捻出するため、以下のどちらかを選択する。
1. **「更なる美しさ」の追求:** 浮いた星屑でより豪華な装飾を加え、さらなる富（結婚相手、領地）を得る。
2. **「人件費の削減」:** お針子の工夫によって星屑が浮いたら、そのお針子を解雇し、お針子の給金分で星屑を買い足す。
